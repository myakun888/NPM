# Update dependencies / package

untuk mengupdate dependencies atau package ada dua cara sama seperti saat membuat nya



saat kita ingin mengupdate package kita , hampir sama dengan saat menginstal atau membuat nya

masuk ke package jsonya , ubah ke versi yang kita inginkan
setelah itu ketik perintah `npm update` pada termialnya


,cara lain
jika kita dalam penulisan versi nya mengguanakn tanda `~` atau `^`
kita dapat otomatis keversi yang sesuai dengan tanda fungsi semantic versi tersebut
(untuk lebih jelas tentan tanda semantic versin silahkan baca [[8dependencies_version]] dan  [[informasi semantic]])
kita tinggal jalankan perintah 
```cmd
npm update namelibrary

atau jika ingin semua library di update 

npm update
```
